# 1. 프로젝트 소개

- 반려견 지도 웹 서비스
    1. 반려견 관련 장소(숙소, 식당, 산책 등) 정보 제공
    2. 대형견 출입 가능 여부 및 견종 필터링
    3. 인증된 견주의 생생한 리뷰
