# 1. 프로젝트 소개

- 반려견 지도 웹 개발
    1. 반려견 관련 장소(숙소, 식당, 산책 등) 정보 제공
    2. 대형견 출입 가능 여부 및 견종 필터링
    3. 인증된 견주의 생생한 리뷰

# 2. 기능

### 1) 역할

- 디자이너 : 강연우
- 프론트엔드 : 유하경, 임가은
- 백엔드 : 권형일, 윤상연

### 2) 개발 환경

- 디자인 : Ai, Figma, Ps, Ae
- 프론트엔드 : HTML, CSS, JS, React
- 백엔드 : JAVA, OAUTH, spring, AWS, MySQL, JWT
- 협업 : Notion, GitHub, Figma

### 3) 구현 기능

1. ***애견 관련 장소 추천 지도***
    1. 위치 기반
    2. 지도 내 아이콘 표시 기능 → 클릭 시 상세 정보
    3. 대형, 중형, 소형견 필터 기능
    4. 카테고리
        1. 미용, 애견 용품, 병원, 카페
                
    5. 장소 평점 및 리뷰
        1. 리뷰(300자) + 별점(0.5 단위)
                
    6. 즐겨찾기
    7. 검색 (후순위)
        1. 검색어 포함 장소가 거리순으로 나열
        2. 상세정보 리스트 나열
        3. 지도에 마커 표현
    
2. ***마이 페이지***
    1. 회원 정보 수정
        1. 프로필 사진 설정
            1. 직접 넣기 or 일러스트
    2. 즐겨찾기(찜)
        1. 지도에 별표 표시
    3. 강아지 정보
    
3. ***회원가입 / 로그인***
    1. 간편 로그인 (구글, 네이버)
    2. 회원가입 입력 받을 정보
        1. 아이디 (이메일)
        2. 비밀번호
        3. 닉네임
